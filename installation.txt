fungal_modelseed_pipeline
=========================

This file lists the packages needed for the current minimal pipeline and the
commands to run it from start to finish.

This project currently uses:
- modelseedpy
- pyyaml
- cobra
- pandas
- matplotlib

Also required locally:
- ModelSEEDDatabase
- a protein FASTA input file (.faa)

Example local clone for the database:
- git clone https://github.com/ModelSEED/ModelSEEDDatabase.git


Suggested environment setup
---------------------------

If you want a fresh conda environment:

conda create -n fungal-modelseed python=3.10 -y
conda activate fungal-modelseed
pip install modelseedpy pyyaml cobra pandas matplotlib

If you already have a conda environment, install the same packages there.

Recommended full run
--------------------

Replace these with your own values:
- INPUT_FASTA=ncbi_dataset/data/GCA_000182925.2/protein.faa
- MODEL_ID=my_model

From the project root:

cd /home/constantinos/projects/Biohacathon/fungal_modelseed_pipeline

1. Detect the input type

python scripts/prepare_input.py --input INPUT_FASTA

Example:

python scripts/prepare_input.py --input ncbi_dataset/data/GCA_000182925.2/protein.faa

2. Load the protein FASTA into a ModelSEEDpy genome object

python scripts/first_modelseed_step.py --input INPUT_FASTA

Example:

python scripts/first_modelseed_step.py --input ncbi_dataset/data/GCA_000182925.2/protein.faa

3. Build the draft model with RAST

python scripts/build_draft_model.py --input INPUT_FASTA --model-id MODEL_ID --use-rast

Example:

python scripts/build_draft_model.py --input ncbi_dataset/data/GCA_000182925.2/protein.faa --model-id test_model_rast --use-rast

4. Gapfill and export the draft model

python scripts/gapfill_and_export_model.py --model-dir data/models/MODEL_ID

Example:

python scripts/gapfill_and_export_model.py --model-dir data/models/test_model_rast

5. Inspect the exported model in COBRApy

python scripts/inspect_with_cobra.py --model-dir data/models/MODEL_ID

Example:

python scripts/inspect_with_cobra.py --model-dir data/models/test_model_rast

6. Run the first-pass exchange-based media screen

python scripts/screen_media.py --model-dir data/models/MODEL_ID --media config/media_library.yml

Example:

python scripts/screen_media.py --model-dir data/models/test_model_rast --media config/media_library.yml

7. Diagnose the exchange space

python scripts/diagnose_exchange_space.py --model-dir data/models/MODEL_ID

Example:

python scripts/diagnose_exchange_space.py --model-dir data/models/test_model_rast

8. Save a compact growth-debug report

python scripts/debug_growth.py --model-dir data/models/MODEL_ID

Example:

python scripts/debug_growth.py --model-dir data/models/test_model_rast

9. Run the oracle-growth debug check

python scripts/run_oracle_growth.py --model-dir data/models/MODEL_ID

Example:

python scripts/run_oracle_growth.py --model-dir data/models/test_model_rast

10. Screen oracle-derived debug media

python scripts/screen_oracle_medium.py --model-dir data/models/MODEL_ID

Example:

python scripts/screen_oracle_medium.py --model-dir data/models/test_model_rast

11. Benchmark bio2 rate and the debug yield proxy across those conditions

python scripts/benchmark_bio2.py --model-dir data/models/MODEL_ID

Example:

python scripts/benchmark_bio2.py --model-dir data/models/test_model_rast

12. Inspect one ranked debug condition in detail

python scripts/inspect_oracle_condition.py --model-dir data/models/MODEL_ID --condition central_carbon_precursors

Example:

python scripts/inspect_oracle_condition.py --model-dir data/models/test_model_rast --condition central_carbon_precursors
