# fungal_modelseed_pipeline

Minimal local pipeline for fungal draft metabolic-model work with ModelSEEDpy and COBRApy.

Current milestone:
- build a draft model from protein FASTA
- export it for COBRApy
- benchmark a small set of draft growth conditions
- run optional validation checks on the draft model, including a theoretical upper-bound mode

Planned future additions:
- more realistic media benchmarking
- targeted product benchmarking
- UI
